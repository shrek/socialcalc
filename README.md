This is an example implementation of a Scala JVM based server backend.

It Uses:

webserver   -- Jetty 
Templating  -- Freemarker
Language    -- Scala/Java

datastore   -- choice of:
                     MySql or other DB (using Squeryl)
                     Cassandra
                     Direct File Io


Dependencies:
-------------

java sdk
scala 
sbt
